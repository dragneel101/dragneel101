## 🚀 Featured Project

### [SupportDesk](https://github.com/dragneel101/ticketingSystem) — Customer Support Ticketing Platform

A full-featured internal ticketing system built for real support teams.

- 🎫 Ticket CRUD with status, priority & assignee management  
- 🔐 Secure session auth (bcrypt + rolling sessions)  
- 📋 Complete audit trail — every change logged with actor & timestamp  
- 📝 Internal agent notes with role-based visibility  
- 📊 Live dashboard — open/pending/resolved/unassigned metrics  
- 👤 Customer record auto-creation on ticket submission  
- 🧪 Backend tested with Jest + Supertest

**Stack:** React 19 · Vite · Express · PostgreSQL · CSS custom properties
